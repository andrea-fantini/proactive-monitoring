# Proactive Monitoring Presentation

A Marp-based presentation on shifting from reactive to proactive monitoring strategies. View the slides here: **[https://andrea-fantini.github.io/proactive_monitoring/](https://andrea-fantini.github.io/proactive_monitoring/)**

## About This Project

This presentation covers:
- Problem analysis and stakeholder alignment
- Phased MVP rollout strategy
- Outcomes and impact metrics
- Key lessons learned

## Built With

- [Marp](https://marp.app/) - Markdown presentation framework
- GitHub Pages for hosting

## Local Development

1. Install [Visual Studio Code](https://code.visualstudio.com/)
2. Install the [Marp for VS Code extension](https://marketplace.visualstudio.com/items?itemName=marp-team.marp-vscode)
3. Open `slides/Slides.md` to preview and edit

## Template Reference

This project uses the [Marp Slides Template](https://github.com/codebytes/marp-slides-template). To start a new presentation project, use that template.
